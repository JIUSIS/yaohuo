# DCloud Android AAR libraries

Copy the required DCloud Android offline SDK AAR files into this directory before building the Android project.

Required base files:

```text
lib.5plus.base-release.aar
uniapp-v8-release.aar
android-gif-drawable-1.2.29.aar
oaid_sdk_1.0.25.aar
breakpad-build-release.aar
utsplugin-release.aar
```

These third-party binary SDK files are intentionally not committed to this repository. Download the Android offline SDK from DCloud and copy the matching files for your HBuilderX version.
